Goals

  • Image for a tweet about #ECMAScript 2049
  • Use the Blade Runner font: https://www.dafont.com/blade-runner.font
    • Download to blade_runner.zip
    • Expand BLADRMF_.TTF
    • Load the ttf using a style tag, type attribute default is text/css
    • How to load a font: https://www.webdesignerdepot.com/2013/01/how-to-use-any-font-you-like-with-css3/
  • Largest possible original fetched from: http://www.geeksundergrace.com/wp-content/uploads/2017/10/blade-runner-2049-2017-lu.jpg
    • Size is 8392x4721
    • Output image size: 440x220 and 2:1 ratio
      https://sproutsocial.com/insights/social-media-image-sizes-guide/#twitter
    • But use maximum allowed size of 1024x512
  • Superimpose text on top of image
    • Use flex layout
      https://stackoverflow.com/questions/42450098/positioning-text-over-image-html-css
    • Image is background of parent div
    • Text is in child div with roght float and padding
